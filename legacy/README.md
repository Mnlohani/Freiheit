# Legacy Code — v1.0

These files are from the original pip-based version of Freiheit.
They are kept for reference to show the evolution of the architecture.

- `llm_app.py` — original single-file app with LangChain + TTS
- `llm_with_nn_app.py` — version with DistanceNN Vision Transformer integration

Current version uses FastAPI + Streamlit in `src/backend` and `src/frontend`.
